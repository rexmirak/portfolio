A Self-Adaptive Agentic Moving Target Defense Architecture

Cybersecurity AI & Machine Learning Master's Thesis View on GitHub

An intelligent intrusion prevention system combining reinforcement learning agents with moving target defense for real-time, autonomous cyber threat response.

Problem & Solution

The Problem

Traditional intrusion prevention systems rely on static rulesets and manual response workflows, creating response delays measured in hours or days. Modern adversaries exploit this lag, moving laterally through networks before defenders react. Static defenses become predictable, allowing attackers to probe and map attack surfaces over time. Security teams face alert fatigue from high false positive rates, while critical threats slip through during triage backlogs.

The Solution

This thesis presents a self-adaptive agentic architecture combining reinforcement learning with moving target defense (MTD) principles. Autonomous agents analyze security alerts in real-time, make triage decisions without human intervention, and dynamically reconfigure system defenses to create an unpredictable attack surface. The system learns optimal response strategies through Q-learning, continuously adapting its defense posture based on observed attack patterns while maintaining operational availability.

Research Impact

This thesis bridges the gap between AI research and practical cybersecurity operations, demonstrating how reinforcement learning can automate traditionally manual security workflows. The architecture addresses a critical industry need: organizations facing sophisticated persistent threats require systems that adapt faster than human analysts can respond.

By combining agentic AI with moving target defense, the work contributes to the emerging field of autonomous cyber defense, where systems proactively reshape their attack surfaces rather than reactively responding to detected intrusions.
